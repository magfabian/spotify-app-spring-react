## Spotify App

This application based on Spotify API. On this website you can browse new releases, search tracks, create your own playlists and etc. This was our school final project. 

Technologies what we used:
 - React
 - Spring Boot
 - Spring Security
 - Context API
 - React Router
 - Semantic UI
 - Hibernate
